Detailed Description

Functionality for calculating 3D panning and Ambisonics coefficients for a point source.

While this is useful for rendering point sources on surround speakers, the main advantage of Phonon's Panning Effect is that it can pan a point source and generate Ambisonics data.

Creates a Panning Effect object.

This can be used to render a point source on surround speakers, or using Ambisonics.

Parameters
rendererHandle to a Binaural Renderer object.
inputFormatThe format of the audio buffers that will be passed as input to this effect. All subsequent calls to iplApplyPanningEffect for this effect object must use IPLAudioBuffer objects with the same format as specified here. The input format must not be Ambisonics.
outputFormatThe format of the audio buffers which will be used to retrieve the output from this effect. All subsequent calls to iplApplyPanningEffect for this effect object must use IPLAudioBuffer objects with the same format as specified here. Any valid audio format may be specified as the output format.
effect[out] Handle to the created Panning Effect object.
Returns
Status code indicating whether or not the operation succeeded.

IPLAPI IPLvoid iplApplyPanningEffect ( IPLhandle  effect,
IPLAudioBuffer  inputAudio,
IPLVector3  direction,

Applies 3D panning to a buffer of audio data, using the configuration of a Panning Effect object.

The input audio is treated as emanating from a single point. If the input audio buffer contains more than one channel, it will automatically be downmixed to mono.

Parameters
effectHandle to a Panning Effect object.
inputAudioAudio buffer containing the data to render using 3D panning. The format of this buffer must match the inputFormat parameter passed to iplCreatePanningEffect.
directionUnit vector from the listener to the point source, relative to the listener's coordinate system.
outputAudioAudio buffer that should contain the rendered audio data. The format of this buffer must match the outputFormat parameter passed to iplCreatePanningEffect.

This is useful if the Panning Effect object is going to be disabled/unused for a few frames; resetting the internal state will prevent an audible glitch when the Panning Effect object is re-enabled at a later time.
