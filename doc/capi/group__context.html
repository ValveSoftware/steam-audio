Detailed Description

Defines a Context object, which controls low-level operations of Phonon.

Typically, a Context is specified once during the execution of the client program, before calling any other API functions. Once any API function is called, changing the Context may lead to undefined behavior.

◆ iplDestroyContext()

IPLAPI IPLvoid iplDestroyContext ( IPLhandle *  context)

Destroys a Context object.

If any other API objects are still referencing the Context object, it will not be destroyed; destruction occurs when the Context object's reference count reaches zero.

Parameters
context[in, out] Address of a handle to the Context object to destroy.
