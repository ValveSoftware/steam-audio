Detailed Description

The format of an audio buffer.

Whenever you pass audio data to or from Phonon, you must describe the format in which the audio is encoded. Phonon only supports uncompressed PCM wave data, stored in 32-bit floating point format. However, Phonon supports many different multi-channel and Ambisonics formats, and the IPLAudioFormat tells Phonon how to interpret a buffer of audio data.
