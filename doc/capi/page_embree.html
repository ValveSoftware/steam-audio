Embree Support

Steam Audio provides optional support for Intel Embree, which is a highly optimized ray tracer that uses high-performance algorithms and Single Instruction Multiple Data (SIMD) instruction sets of modern x86_64 CPUs.

Embree works with a wide range of x86_64 CPUs.

To configure Steam Audio to use Embree for sound propagation, you must create an IPLSimulationSettings struct, with its sceneType parameter set to IPL_SCENETYPE_EMBREE. This IPLSimulationSettings struct must be passed to the iplCreateScene and iplCreateEnvironment functions.
